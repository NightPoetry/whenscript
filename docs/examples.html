特性测试样例:一屏一个概念

语言核心不需要浏览器——下面十个最小可跑片段,覆盖类型、派生 :=、反应式家族(when/changed)、event 委托、group、字符串插值、定点小数、each/break,以及 function 一等类型。点「在 Playground 跑」当场复现。

变量与类型

强类型直给:number / string / bool,运算即所见。

派生绑定 :=

total 持续等于 price + shipping:只改输入,派生值自己变。

02-derive.when ▶ 在 Playground 跑 
import { print } from "console";
number price = 100;
number shipping = 8;
number total := price + shipping;  // total 持续等于右边

print("total = ", total);
price = 200;                       // 只写 price,total 自己变
print("total = ", total);
shipping = 20;
print("total = ", total);
output · 真实运行结果
total = 108
total = 208
total = 220

when · 边沿触发

条件从假变真的那一刻触发一次;持续为真不重复刷屏。

03-when-edge.when ▶ 在 Playground 跑 
import { print } from "console";
number count = 0;

when(count == 3) { print("count 到 3 了!"); }
when(count > 4)  { print("超过 4: ", count); }

count = 1;
count = 3;   // 触发第一条
count = 5;   // 触发第二条
count = 6;   // 持续为真,不重复触发(边沿语义)
output · 真实运行结果
count 到 3 了!
超过 4: 5

changed · 每变必报

changed 记流水账,when 只在越线那一下告警——各司其职。

04-changed.when ▶ 在 Playground 跑 
import { print } from "console";
number temp = 20;

changed(temp) with (old, new) {
  print("温度 ", old, " -> ", new);
}
when(temp >= 26) {
  print(">> 达到 26 度,告警一次");
}

temp = 23;
temp = 26;
temp = 28;   // changed 照报,when 不再响
output · 真实运行结果
温度 20 -> 23
温度 23 -> 26
>> 达到 26 度,告警一次
温度 26 -> 28

event 委托

event<T> 是一等值:emit 触发、with 收参,信号线还能交给函数。

05-event.when ▶ 在 Playground 跑 
import { print } from "console";
event<number, string> onLog;

when(onLog) with(number code, string msg) {
  print("[", code, "] ", msg);
}

emit onLog(200, "OK");
emit onLog(404, "Not Found");

// 委托可传递: 把信号线交给函数, 让它来发射
void worker(event<number, string> report) {
  emit report(500, "worker 内部出错");
}
worker(onLog);
output · 真实运行结果
[200] OK
[404] Not Found
[500] worker 内部出错

group 数据结构

数组和哈希表的统一体:数字键走数组,字符串键走哈希。

06-group.when ▶ 在 Playground 跑 
import { print } from "console";
group arr = [1, 2, 3];
print("arr[0] = ", arr[0]);
arr[0] = 100;
print("改后 arr[0] = ", arr[0]);

// 字符串键自动走哈希表, 数字键走数组
group user = ["name": "when", "age": 3];
print("user 的 name = ", user["name"]);
print("user 的 age  = ", user["age"]);
output · 真实运行结果
arr[0] = 1
改后 arr[0] = 100
user 的 name = when
user 的 age = 3

定点小数 number<string>

浮点的 0.1 + 0.2 会漂移;number<string> 精确定点,:2 定标两位小数。

08-money.when ▶ 在 Playground 跑 
import { print } from "console";
// 浮点的 0.1 + 0.2 有名地不等于 0.3
// number<string> 是精确定点; :2 = 结果保留两位小数
number<string> a = "0.10";
number<string> b = "0.20";
number<string>:2 sum := a + b;   // 派生和, 两位小数

print("浮点: 0.1 + 0.2 = ", 0.1 + 0.2);
print("定点: a + b     = ", sum);
a = "99.90";
print("a 一变, sum 跟着变: ", sum);
output · 真实运行结果
浮点: 0.1 + 0.2 = 0.30000000000000004
定点: a + b = 0.30
a 一变, sum 跟着变: 100.10

each · break 全量跳出

each(group) 遍历,break 撞到即刻整体跳出循环。

09-each.when ▶ 在 Playground 跑 
import { print } from "console";
group nums = [3, 1, 4, 1, 5, 9, 2, 6];
number sum = 0;

each(nums) with(n) {
  if (n == 9) { break; }   // 撞到 9,全量跳出循环
  sum = sum + n;
}
print("累加到 9 之前: sum = ", sum);
output · 真实运行结果
累加到 9 之前: sum = 14

function 是一等类型

函数能装进变量再调用——function 是变量类型,不是保留字。

10-function.when ▶ 在 Playground 跑 
import { print } from "console";

number add(number a, number b) { return a + b; }
number square(number n) { return n * n; }

print("add(3, 4)  = ", add(3, 4));
print("square(5)  = ", square(5));

// function 是一等类型: 装进变量再调用
function op = add;
print("op(10, 20) = ", op(10, 20));
output · 真实运行结果
add(3, 4) = 7
square(5) = 25
op(10, 20) = 30
