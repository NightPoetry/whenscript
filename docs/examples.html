派生绑定 :=

total 持续等于 price + shipping:只改输入,派生值自己变。

02-derive.when ▶ 在 Playground 跑 
import { print } from "console";
number price = 100;
number shipping = 8;
number total := price + shipping;  // total 持续等于右边

print("total = ", total);
price = 200;                       // 只写 price,total 自己变
print("total = ", total);
shipping = 20;
print("total = ", total);
output · 真实运行结果
total = 108
total = 208
total = 220

when · 边沿触发

条件从假变真的那一刻触发一次;持续为真不重复刷屏。

03-when-edge.when ▶ 在 Playground 跑 
import { print } from "console";
number count = 0;

when(count == 3) { print("count 到 3 了!"); }
when(count > 4)  { print("超过 4: ", count); }

count = 1;
count = 3;   // 触发第一条
count = 5;   // 触发第二条
count = 6;   // 持续为真,不重复触发(边沿语义)
output · 真实运行结果
count 到 3 了!
超过 4: 5

changed · 每变必报

changed 记流水账,when 只在越线那一下告警——各司其职。

04-changed.when ▶ 在 Playground 跑 
import { print } from "console";
number temp = 20;

changed(temp) with (old, new) {
  print("温度 ", old, " -> ", new);
}
when(temp >= 26) {
  print(">> 达到 26 度,告警一次");
}

temp = 23;
temp = 26;
temp = 28;   // changed 照报,when 不再响
output · 真实运行结果
温度 20 -> 23
温度 23 -> 26
>> 达到 26 度,告警一次
温度 26 -> 28

event 委托

event<T> 是一等值:emit 触发、with 收参,信号线还能交给函数。

05-event.when ▶ 在 Playground 跑 
import { print } from "console";
event<number, string> onLog;

when(onLog) with(number code, string msg) {
  print("[", code, "] ", msg);
}

emit onLog(200, "OK");
emit onLog(404, "Not Found");

// 委托可传递: 把信号线交给函数, 让它来发射
void worker(event<number, string> report) {
  emit report(500, "worker 内部出错");
}
worker(onLog);
output · 真实运行结果
[200] OK
[404] Not Found
[500] worker 内部出错

group 数据结构

数组和哈希表的统一体:数字键走数组,字符串键走哈希。

06-group.when ▶ 在 Playground 跑 
import { print } from "console";
group arr = [1, 2, 3];
print("arr[0] = ", arr[0]);
arr[0] = 100;
print("改后 arr[0] = ", arr[0]);

// 字符串键自动走哈希表, 数字键走数组
group user = ["name": "when", "age": 3];
print("user 的 name = ", user["name"]);
print("user 的 age  = ", user["age"]);
output · 真实运行结果
arr[0] = 1
改后 arr[0] = 100
user 的 name = when
user 的 age = 3

定点小数 number<string>

浮点的 0.1 + 0.2 会漂移;number<string> 精确定点,:2 定标两位小数。

08-money.when ▶ 在 Playground 跑 
import { print } from "console";
// 浮点的 0.1 + 0.2 有名地不等于 0.3
// number<string> 是精确定点; :2 = 结果保留两位小数
number<string> a = "0.10";
number<string> b = "0.20";
number<string>:2 sum := a + b;   // 派生和, 两位小数

print("浮点: 0.1 + 0.2 = ", 0.1 + 0.2);
print("定点: a + b     = ", sum);
a = "99.90";
print("a 一变, sum 跟着变: ", sum);
output · 真实运行结果
浮点: 0.1 + 0.2 = 0.30000000000000004
定点: a + b = 0.30
a 一变, sum 跟着变: 100.10
playground · 页面与游戏

game.when

Flappy 金苹果

一个完整的游戏:没有游戏引擎、没有主循环对象、没有一个监听器的注册/解绑。碰撞检测、计分、柱子回收,全是扁平的 when 条件;跳跃和物理冻结是代数,不是分支。点击画面拍打翅膀……呃,拍打苹果。

在 Playground 打开 → neon.when

光扫边框

图层模型实战:Stack 图层组 + mask 遮罩 + spin 动效。conic 渐变光层被裁成 2px 边框环,转动的是渐变角而不是盒子;另附文字光扫。

在 Playground 打开 → motion.when

动画编排

Web Animations API 原语 animate:关键帧和时序都是普通 group,序列化后交给浏览器原生动画引擎——delay 错峰入场,弹簧式缓动回弹。

在 Playground 打开 → events.when

富事件跟随

类型即契约:声明 event<group>,DOM 事件的坐标/键码/修饰键就装成 group 喂进来。聚光灯跟鼠标、hover 是反应式状态、键盘面板实时显示按键。

在 Playground 打开 → styled.when

反应式购物车

在 Playground 打开 → switch.when
