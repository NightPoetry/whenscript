01 · types基础类型与变量声明就是「类型 名字 = 值」。number 统一整数与浮点(内部有 int < float < 高精度的精度阶,计算「升-算-降」不打扰你,赋值才是强制点);string 支持 ${...} 插值;group 是数组和哈希表的统一体。要精确小数(比如钱),用定点 number<string>,冒号配置位 :2 把结果定标到两位小数。
import { print } from "console";
number n = 42; // 整数与浮点同属 number(精度阶 int < float < 高精度)
number pi = 3.14;
string msg = "Hello";
bool ok = true;
group user = ["name": "when", "age": 3]; // group: 数组 + 哈希表的统一体
print(msg, ", ", user["name"], "!");
print("n * pi = ", n * pi);
print("ok = ", ok);

02 · reactive反应式家族:when · changed · once · :=四个成员各管一件事:when(cond) 在条件从假变真那一刻触发一次(边沿,不刷屏);changed(x) 每次变化都报,还给你旧值新值;once 一辈子只响一次;b := expr 让 b 持续等于右边,像电子表格的单元格。每次写入,引擎先让所有 := 派生收敛到不动点,再判 when——反应体里读到的永远是一致的世界(glitch-free)。
import { print } from "console";
number count = 0;
number double := count * 2; // := 持续绑定: count 一变它就变
when(count == 3) { print("到 3 了(when: 条件变真那一下)"); }
once(count > 4) { print("只响一次(once)"); }
changed(count) with (old, new) {
print("count ", old, " -> ", new, ", double = ", double);
}
count = 3;
count = 5;
count = 2;
count = 6; // once 已用掉, 不再响

03 · eventsevent 委托event<T> 声明一根带类型的信号线:emit 往里发,when(事件) with(带类型的参数) 收。事件是一等值——可以当参数传给函数、当返回值带出来、经 import 跨模块共享同一根线。
import { print } from "console";
event<number, string> onLog;
when(onLog) with(number code, string msg) {
print("[", code, "] ", msg);
}
emit onLog(200, "OK");
emit onLog(404, "Not Found");
// 委托可传递: 把信号线交给函数, 让它来发射
void worker(event<number, string> report) {
emit report(500, "worker 内部出错");
}
worker(onLog);

04 · entitygroup 实体与 until给 group 的花括号体里写上 when/changed,它就成了实体:状态和行为住在一起,不需要 class。delete 名字 一步回收整个实体连同它的全部反应;until(cond) 更进一步——声明生命周期,条件一到,实体自动回收,确定性的,没有垃圾回收器猜。
import { print } from "console";
bool closed = false;
group cart = {
cart.total = 0;
changed(cart.total) { print("[cart] 总价 ", cart.total); }
when(cart.total > 100) { print("[cart] 包邮解锁!"); }
} until(closed);
cart.total = 40;
cart.total = 120;
closed = true; // until 边沿 → 实体连同它的反应一起确定性回收
cart.total = 999; // 静默: cart 的 when 已随实体回收

05 · functions函数与 with函数声明以返回类型开头:number add(number a, number b) { ... },无返回值用 void。事件处理器的参数有两种写法——with(string m) { ... } 和箭头 (string m) => { ... },完全等价(箭头就是 with 的简写);两种写法的参数都要带类型。
import { print } from "console";
number add(number a, number b) { return a + b; }
print("add(10, 20) = ", add(10, 20));
// 事件参数两种写法完全等价: with 与箭头(参数都带类型)
event<string> onMsg;
when(onMsg) with(string m) { print("with 收到: ", m); }
when(onMsg) (string m) => { print("箭头收到: ", m); }
emit onMsg("hi");

06 · modules模块 import / export文件末尾 export { 名字们 }; 列出要暴露的函数、变量、事件;另一个文件 import { ... } from "./路径.when"; 拿来用。经 import 共享的同一个 event 跨模块完全互通——这是设计好的共享机制,不是巧合。
lib/math.when
number square(number n) { return n * n; }
number PI = 3.14159;
export { square, PI };
main.when
import { print } from "console";
import { square, PI } from "./lib/math.when";
print("square(5) = ", square(5));
print("PI = ", PI);
多文件示例需要本地 CLI(Playground 是单文件的):把两个文件放好后 whenscript main.when。
07 · frontend前端容器语法一切皆容器:container 名字() { ... } 声明组件,统一块 { 配置: 值; 子元素 } 既写样式也放孩子。给按钮起个名,when(按钮) 直接就是点击反应;文本里的 ${...} 插值自动跟着数据变——布局就是 := 电路,没有虚拟 DOM。
import { Text, Button } from "dom";
number count = 0;
number double := count * 2; // 布局即电路: 值变, 界面自己动
container App() {
padding: "24px"; fontFamily: "system-ui, sans-serif";
Text("count = ${count} · double = ${double}") { fontSize: "20px"; weight: bold }
add = Button("+1") { padding: "8px 16px"; cursor: "pointer" }
when(add) { count = count + 1; }
}
App();

08 · toolchain工具链命令一个 whenscript 二进制,全套工具面。工具面能比执行面报得更多、看得更多,但绝不改执行语义。
命令用途
whenscript <file.when>运行程序;语法/语义错在执行前响亮报错,不静默吞。
whenscript check [--json] <file>静态检查(lex → parse → 语义),一次报出全部语义错误,带行列号,不执行。
whenscript fmt [--write|--check] <file>保真格式化:token 流保真、注释一个不丢、幂等;--check 供 CI 用。
whenscript pkg <install|add|remove|list>去中心化包管理:when.json + when.lock + when_modules/,支持 git 与本地路径依赖,无安装脚本。
whenscript graph <file> [-o out.html]导出运行时反应式依赖图的只读快照(交互 HTML,也可 --dot/--json)。
whenscript repl交互式会话,回放式地试语言。
whenscript lsp在 stdio 上说 LSP,供编辑器接入:诊断、悬停中文文档、补全。
